Lives LPK_Lives.cs

This component can be added to any object to define its amount of lives and dispatches a LPK_OutOfLives event upon running out of them.

Suggested use:

Use with LPK_Modify_Health_On_Event or LPK_Kill_On_Event to subtract lives when health reaches 0, or when a specific event happens (ex: collision with an enemy or environment).

Display the lives easily using LPK_Text_Display and a LivesText tag.
