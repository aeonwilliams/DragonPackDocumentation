Dispatch on Start LPK_DispatchOnStart.cs

Activates the OnEvent function on other scripts when this object calls it's start function.

Suggested use:

Use as an alternate way to make game behavior happen automatically on an object's spawn rather than using a timer.

Examples found in: Project 3
