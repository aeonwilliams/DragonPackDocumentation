Detach on Event LPK_DetachOnEvent.cs

Detach an object from all objects it is a child to.

Suggested use:

Use with Attach on Event to alter the hierarchy.

Caveats:

  • No auto-detaching happens behind the scenes.
  • If an object is a child to a child x deep into the hierarchy, and this component is used, it will be detached from ALL objects.
  • If an object with children is detached from an object, it's relationship with it's children will stay the same.

Examples found in: Project 4
