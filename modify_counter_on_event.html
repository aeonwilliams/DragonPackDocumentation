Modify Counter on Event LPK_ModifyCounterOnEvent.cs

This component can be added to any object to cause it to dispatch an event to modify a target counter upon receiving an event.

Suggested use:

Connect to 1 or more counter components to modify the value on the counter(s). This component implements the event system, so it can be on the same object as the counter(s) it's connected to, or any other object in the game. Use the cooldown element to delay the modification of the counter if needed.

Note: To modify multiple counters on the same object, you need to drag the component into the Counters section of this component. Using Unity's UI selector will not work.

Examples found in: Project 1
