Aggressive Path Follower LPK_AggressivePathFollower.cs

This component causes its game object to follow a pre-determined path by the designer. When a chosen object is within a defined range of the owner, it will leave the path and either chase or run from the object.

Suggested use:

Create a path of game object nodes, and assign them in the correct component property location. Create an object (enemy, pickup, etc) to either chase or run away from an approaching object, like the player.

Examples found in: Project 4

Component Properties
Loop Type
LPK_PathFollowerLoopType
Default: SINGLE
How to handle path looping once the end of the path is reached.
  • SINGLE: object will follow the path once, and stop on the last node.
  • LOOP: object will follow the path, and upon reaching the last node, will take the shortest route to the first node to begin following the path again.
  • LOOP_TELEPORT: object will follow the path, and upon reaching the last node, will teleport to the first node to begin following the path again.
  • LOOP_BACKTRACK: object will follow the path, and upon reaching the last node, will go in reverse order through the list of nodes.
Speed
Float
Default: 5
Speed (units per second) to move along the path.
Aggression Properties
Enemies
Array of Objects
Default: array size = 0, array elements = None (Transform)
Objects to attack/run from when they are within range.

Enemy Tag
Array of Strings
Default: array size = 0, array elements = untagged
Tag enemies will have. Useful if this path follower should react to multiple different GameObjects.

Aggression Range
Float
Default: 10
Relative distance to the owner at which the tracking object will change behavior.

Aggression Speed
Float
Default: 5
Speed (units per second) the tracking object will move at while within range of the owner.

Is Coward
Boolean
Default: False
False means the tracking objects will run towards the owner, true means they will run from the owner.

Don't Forget
Boolean
Default: False
If this is true, the tracking object(s) will follow the owner forever once the range threshold is reached. Otherwise, once the owner is out of range, the tracking object(s) will take the shortest route back to their path, and continue following it.
Nodes
Array of Objects
Default: array size = 0, array elements = None (Transform)
Nodes that make up the path to follow. They can be any Unity object, as long as it has a Transform component.

PathFollowerLostEnemyEvent
LPK_EventSendingInfo

Default Event Sending Mode: ALL

PathFollowerReachNodeEvent
LPK_EventSendingInfo
Default Event: None (LPK_Event Object)

PathFollowerReachFinalNodeEvent
LPK_EventSendingInfo

Default Event Sending Mode: ALL
Event sent when the path follower reaches the final node in its current path.
