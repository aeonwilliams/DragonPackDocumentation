Counter LPK_Counter.cs

This component can be added to any object to have it track a value relevant to the game. That value might be a collectible, currency, score, etc.

Suggested use:

Use on the same object as Modify Counter on Event to change the value of the counter on the given event - use the counter to dispatch events once it reaches a certain value or threshold. Use a Text Display to easily display the value of the counter.

Examples found in: Project 1 - Score
