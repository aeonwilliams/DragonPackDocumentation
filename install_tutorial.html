How to Install DragonPack

Before you can begin making games using DragonPack, you will need to install the required components. Installation is not difficult, but must be done correctly to function.

Downloading the Files

  • Step 1

    The first thing you need to do is download the .zip file containing DragonPack.

    If you are a teacher or student using this for a specific class, download DragonPack from the specified Canvas or Moodle site, as seen below:


  • Step 2

    Once the file is downloaded, extract the zip. Now you have a zipped version, and an unzipped version!



    You will need to perform the following steps every time you start a new project using the Dragon Pack, so you should move the folder to a safe location.

Installing the Components

  • Step 1

    Create a new project in Unity.



  • Step 2

    From the Assets menu, choose Import Package -> Custom Package.



    A standard directory for your OS will open.
    • Navigate to the DragonPack folder you just downloaded and saved.
    • Select the file called Unity_LPK_Code and click Open.
    • Leave everything selected (this is the default setting) and click Import.

That's It!

Your assets folder should now look like this:



You can now use all DragonPack components, found in the LPK folder.
